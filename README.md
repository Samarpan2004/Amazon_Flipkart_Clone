# Amazon / Flipkart Clone (MERN) - Ready Project

## Quick start

1. Backend
   - cd backend
   - npm install
   - create a .env file (see .env.example)
   - npm run seed
   - npm run dev

2. Frontend
   - cd frontend
   - npm install
   - npm start

Backend runs on http://localhost:5000
Frontend runs on http://localhost:3000
